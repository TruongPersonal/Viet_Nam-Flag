# Viet Nam Flag

A simple Python program to draw the **Vietnamese flag** using the `turtle` module. Click anywhere on the window to draw the flag with a star and a flagpole.

## Features

- Draws a red rectangle with a yellow five-pointed star.
- Adds shadow effects for depth.
- Includes a flagpole with base.
- Scales automatically based on screen size.

![Vietnamese Flag](screenshot.png)

## How to Run

1. Install Python (>=3.7 recommended).  
2. Run the script:
- Windows
```bash
python main.py
```
- MacOS
```bash
python3 main.py
```
3.	Click anywhere in the window to draw the flag.

## Dependencies
- Python standard library (turtle)
